# AmalCare

AmalCare est une plateforme digitale qui aide les patients atteints de cancer au Maroc à :

* Trouver des centres d'oncologie
* Accéder à des associations de soutien
* Comprendre leur parcours de soins
* Visualiser les services via une carte interactive

## Fonctionnalités

* Carte interactive (Leaflet)
* Filtres de recherche
* Multilingue (FR / AR)
* Données structurées (centres & associations)

## Technologies

* HTML / CSS / JavaScript
* Leaflet.js (cartographie)
* Données JSON locales

## Déploiement

Ce site est hébergé via GitHub Pages.
