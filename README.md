# AmalCare

Plateforme sociale marocaine d’orientation pour les patients atteints de cancer.

## Fonctionnalités

- Centres d’oncologie
- Associations et Maisons de Vie
- Informations assurances
- Version bilingue FR / AR

## Déploiement GitHub Pages

1. Créer un repository GitHub
2. Ajouter les fichiers
3. Aller dans Settings > Pages
4. Choisir branch main
5. Save
